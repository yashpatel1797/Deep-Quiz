# Deep-Quiz

This quiz app will check your knowledge on famous hollywood directors and their films.

## Run Locally

Clone the project

```bash
  git clone https://github.com/yashpatel1797/Deep-Quiz.git
```

Go to the project directory

```bash
  cd Deep-Quiz
```

Open projects

```bash
    index.html
```

Run code.

## Contains

- Homepage
- Category page
- Rules page
- Questions page
- Result page
- Authentication page

## Tech Stack

**Client:** HTML, CSS, Javascript

**Hosted:** Netlify
